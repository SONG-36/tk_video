# 页面设计

> 本文档来源于 `00_design.md`，仅用于阅读，不是权威源。来源：第 7 章；冲突时以 `00_design.md` 为准。

## 页面结构与顺序

商品列表 → 新建商品 → 类目确认 → 型号 / SKU 录入 → 型号差异判断 → 本次拍摄型号选择 → 型号级商品画像补全 → 型号级判断报告 → 材料检查 → 型号级产品图片资产 → 创意方向设定 → 型号级视频批次规划 → 镜头资产规划 → Codex 任务包导出 → 输出结果查看。

## 页面职责摘要

- 商品列表页：展示商品名称、平台、市场、类目、分类、状态、缺失材料数、更新时间和操作。
- 新建商品页：收集产品名称、目标平台、市场、业务阶段及选填资料，通过后输出系列基础信息。
- 类目确认页：展示候选类目、置信度、理由和风险；平台类目必须人工确认。
- 型号 / SKU 录入页：收集型号名、SKU、属性、参数、配件、价格和图片，至少一个型号。
- 型号差异判断页：并排比较共享字段与隔离字段，判断是否分型号制作。
- 目标型号选择页：选择批次唯一目标型号，并持续显示系列、型号和 SKU。
- 型号级商品画像补全页：用结构化选择补齐体积、真人、场景、功能、审美、对比、包装、风险和卖点，检查冲突。
- 型号级判断报告页：整合类目、分类、内容难度、分型、打法、风险、缺失信息及下一步。
- 材料检查页：按必须、建议、风险材料展示缺失项并控制闸门。
- 型号级产品图片资产页：上传和标注图片，计算完整度、缺图与生成能力降级。
- 创意方向页：收集风格、剧情、强度、真人限制、卖点和自由想法，生成专业镜头语言翻译并确认。
- 型号级视频批次规划页：规划 3—5 条视频的目标、分型、卖点、场景、创意和 A/B/C 生成等级。
- 镜头资产规划页：为批次视频生成 Creative Board、Storyboard、Shot Card 和镜头级提示词计划。
- Codex 任务包导出页：将前序结果写为标准任务文件，不直接调用 Codex 插件。
- 输出结果查看页：按批次读取画板、脚本、故事板、镜头卡、提示词、输入计划、剪辑计划、质检和缺失素材。

每个页面的详细字段、输出、状态依赖和通过条件见 `frontend/pages/*/README.md`。
