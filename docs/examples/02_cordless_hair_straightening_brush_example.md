# 无线直发梳示例

> 本文档来源于 `00_design.md`，仅用于阅读，不是权威源。来源：第 21 章。

目标为 TikTok Shop US，美国市场，准备拍视频；类目 `Beauty & Personal Care`，分类小件标品。系列选择黑色无线款 `sku_001`，不同温度档位或配件版本不得混入。

画像：手持小件、最好真人出镜、不需大场景、功能明确、强依赖效果、适合对比、电子、接触人体且含电池。主力分型为买家卖家秀和卖点讲解，推荐测评对比，生活场景辅助，开箱弱推荐。

必须展示真人头发使用前、中、后，以及产品正侧面、梳齿、温控、充电口、防烫和包装；不能只拍开箱。建议补加热速度、温度档位、续航和防烫设计；电池、电气安全、人体接触和认证资料缺失时写入风险约束。

适用创意可使用“上班前 5 分钟整理发型”的生活场景，镜头以步骤近景、前后对比和产品细节为主；真实性与安全约束优先于创意表达。
