# 车载吸尘器完整链路示例

> 本文档来源于 `00_design.md`，仅用于阅读，不是权威源；如与 `00_design.md` 冲突，以 `00_design.md` 为准。
>
> 来源章节：第 20 章。

## 输入与类目

- 产品：车载吸尘器。
- 平台：TikTok Shop US。
- 市场：美国。
- 阶段：准备拍视频。
- 推荐类目：Automotive & Motorcycle。
- 商品分类：小件标品。

## 型号 / SKU

- 商品系列：车载吸尘器系列。
- 本次拍摄型号：黑色标准款。
- `variant_id`：`sku_001`。
- 其他型号图片和参数：本批次不使用。

## 商品画像与判断报告

画像为小件可手持、不需要真人露脸、需要车内场景、功能明确、适合对比、电子电器且含电池。

判断结果：自制视频推荐高；拍摄容易度 5、种草容易度 4、消费决策容易度 3。

## 视频分型

- 生活场景型：主力。
- 卖点讲解型：主力。
- 测评对比型：主力。
- 开箱直拍型：辅助。
- 买家卖家秀型：可选。

## 材料检查

- 必须：产品名称、平台类目、商品分类、核心卖点、使用场景、视频分型。
- 建议：产品图片、车内实拍、吸力、续航、噪音、尘盒容量、竞品视频。
- 风险：电池信息、认证资料、安全标识。

缺失具体参数和合规资料时不得编造。

## 图片资产

- 完整度：标准。
- 产品失真风险：中。
- 已有：正面、侧面、吸头细节、配件、包装、车内脏污场景。
- 缺失：背面、45 度、座椅缝隙真实使用、清洁前后对比。

## 创意方向

- 风格：搞笑反转 + 好莱坞电影感。
- 剧情：痛点夸张 → 普通工具失败 → 产品英雄登场 → 清洁效果 → 前后对比。
- 强度：中等夸张。
- 真人：只允许手部。
- 优先级：产品清晰度 > 卖点表达 > 用户痛点 > 创意剧情 > 运镜炫技。

## 视频批次

- `batch_id`：`batch_001`。
- 目标：测试不同内容角度。
- 数量：3 条。
- 策略：`1A_1B_1C`。
- `video_01`：生活场景 / 座椅缝隙清洁 / A。
- `video_02`：卖点讲解 / 吸力和便携 / B。
- `video_03`：测评对比 / 普通工具 vs 本产品 / C。

素材充足时才可扩展 `video_04/05` 并使用 `2A_2B_1C`。

## 镜头资产

A 级 `video_01` 为 20 秒、5 个镜头：灾难开场、普通方法失败、产品英雄登场、清洁效果、前后对比。产品出现的镜头绑定对应外观、吸头、场景和前后对比图。

## 任务包与输出

任务包绑定 `p_001 + sku_001 + batch_001`，写入 `data/tasks/p_001_sku_001_batch_001_codex_task.{md,json}`。输出位于 `outputs/p_001/batch_001/video_01/`，包含画板、脚本、故事板、镜头卡、提示词、图片计划、剪辑计划和质检清单。
