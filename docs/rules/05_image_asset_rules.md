# 产品图片资产规则

> 本文档来源于 `00_design.md`，仅用于阅读，不是权威源；如与 `00_design.md` 冲突，以 `00_design.md` 为准。
>
> 来源章节：第 13 章。

## 资产类型

`product_reference` 外观锁定、`product_detail` 细节、`usage_scene` 场景、`before_after` 前后对比、`packaging` 包装、`style_reference` 风格参考、`motion_reference` 运镜/动作参考。

## 最低图片要求

- 普通商品：正面、侧面、背面、45 度、尺寸比例、细节、配件、包装。
- 电器额外：按钮、充电口、显示屏/指示灯、接口/插头、标签铭牌、安全标识、配件组合。
- 使用效果类额外：使用前、使用中、使用后、真实场景、真人/手部/宠物互动。

## 完整度等级

- 少于 3 张：禁止生成镜头级提示词，只能生成脚本和补图清单。
- 3—5 张：允许基础提示词，标记高失真风险。
- 6—8 张：允许标准镜头提示词，提示缺失角度。
- 8 张以上且覆盖多角度、细节、场景：允许高质量镜头级提示词。

## 单项字段

`image_id`、`file_name`、`image_type`、`view_angle`、`usage[]`、`quality`、`required`、`notes`。

## 镜头绑定

每个镜头明确使用图片、各图用途、外观锁定图、场景/构图/动作参考和缺图失真风险。

## 通过条件

图片已上传或明确缺失；每张图完成类型标记；关键图完成用途标记；系统生成完整度、缺图清单和能力限制。
