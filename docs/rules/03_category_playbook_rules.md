# 类目打法规则

> 本文档来源于 `00_design.md`，仅用于阅读，不是权威源。来源：第 11 章“类目打法库”。

- 3C 电子：强功能、演示、便利性和多角度；优先卖点、开箱、测评。
- 日用家纺 / 家电 / 家具：真实场景、使用价值、材质规格和效果；优先场景、卖点、真人展示。
- 运动器械 / 户外：安装、完整演示、稳定、安全和便携；优先场景、卖点、真人展示。
- 服饰 / 鞋 / 珠宝 / 配饰 / 箱包：真人、尺码、细节、搭配和风格；优先真人展示、场景、开箱。
- 美妆个护：真人效果、步骤、前后对比和持续时间；优先真人展示、卖点、测评。
- 食品 / 日用品 / 宠物：数量、过程、真实对象和反应；食品展示包装口味和食用，宠物展示互动、安全、耐用和清洁。

各类目的具体开头方式、必须展示项和覆盖平台类目以第 11 章原文为准。
