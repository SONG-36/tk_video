# 创意迁移规则

> 本文档来源于 `00_design.md` 第 18 章，仅用于阅读，不是权威源；如与 `00_design.md` 冲突，以 `00_design.md` 为准。

## 跨文化适配

进入 TikTok US 前检查人物关系、冲突强度、阶层/性别/羞辱风险、英语自然度、美国生活场景、产品出现合理性和 TikTok Shop CTA。不得直译国内短剧台词、照搬霸总/婆媳/豪门/打脸设定或用冒犯换情绪强度。可以保留反差、误会、先压后爽、失败方法对正确方法、产品救场，并改写为本地可理解的生活冲突。

## 产品适配

检查模式能否自然引出产品、产品能否承担解决方案、是否有画面证明、真人/场景/道具需求、剧情是否压过产品、夸大/合规风险、额外素材和产品失真风险。不能让产品承担中段解决方案的模式只能保留为灵感。

## 创意迁移流程

选择 `viral_pattern_card` → 识别注意力触发 → 提取可迁移元素 → 剔除不可迁移元素 → 匹配当前商品画像、视频分型、图片和材料 → 生成 `product_adapted_pattern` → 检查 TikTok 原生风格、产品清晰度和合规 → 进入批次或镜头计划。

## product_adapted_pattern

对象绑定 `adapted_pattern_id`、`source_pattern_id`、`product_series_id`、`variant_id`、`batch_id`、`video_id`，并描述 `adapted_concept`、`hook_type`、前 3 秒文字/画面、`emotional_trigger`、`product_role`、`proof_scene`、CTA 及 `risk_check`。`risk_check` 至少保留剧情压过产品、演员/场景需求和合规风险。

## 与视频批次规划的关系

批次同时测试分型和创意模式，可选择测试不同爆款模式、情绪触发、前 3 秒机制、本地化方式或剧情强度。同一批次应避免 3 条视频使用同一种前 3 秒机制；逐视频引用 `viral_pattern_ref` 与 `adapted_pattern_ref`。

## 与镜头资产规划的关系

模式必须转成镜头。镜头标明对应情绪、是否推进反转、是否证明产品、是否仅铺垫、铺垫是否过长、产品首次出现时间和证明开始时间。证明场景、CTA 画面、文字与声音计划也必须可执行。

## 与 Codex 任务包的关系

任务包携带所用模式卡和适配结果，并明确只参考模式、情绪与节奏，禁止复制人物、台词、画面、音乐、品牌和剧情细节。产品必须承担解决方案，中段用画面证明，结尾提供 CTA。

## 时间与平衡硬规则

- 产品首次清晰出现不能晚于 8 秒。
- 产品证明不能晚于 12 秒开始。
- 剧情铺垫不能超过产品证明；剧情压过产品时必须降级剧情。

跨镜头时间计算和剧情/产品平衡的确定性判定方式尚未定义，由后续设计和 validators 承接，Schema 不假装完成该校验。

## 禁止事项

不得照抄来源、直译短剧、使用未授权内容、让产品晚于 8 秒出现、只有情绪没有证明、让剧情压过产品、为吸睛制造冒犯或把不适配模式硬套到商品。

## 通过条件

模式卡完整；可/不可迁移元素已区分；跨文化与产品适配已检查；适配对象已生成并绑定五类 ID；确认 8 秒内出现、12 秒内证明、CTA 相关、无版权/文化/合规风险且剧情不压过产品后，才进入正式视频规划。
