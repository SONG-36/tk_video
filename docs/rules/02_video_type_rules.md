# 视频分型规则

> 本文档来源于 `00_design.md`，仅用于阅读，不是权威源；如与 `00_design.md` 冲突，以 `00_design.md` 为准。
>
> 来源章节：第 11 章“视频分型规则”“一级类目与视频分型推荐规则”。

规则针对当前 `selected_variant_id` 推荐，只决定优先级，不决定实际生成数量。

## 五类视频

### 生活场景型

商品与具体生活场景结合。要求同一场景下商品不同视角至少出现 2 次，或同一商品出现在至少 2 个使用场景。适合汽车、家居、宠物、户外、厨房、日用家电和清洁工具。

### 卖点讲解型

围绕痛点、功能、优势、价格或优惠展开，商品是主体，画面、声音或字幕至少说明 2 类干货信息。适合功能小件、3C、工具、车载用品、小家电和美容个护电器。

### 开箱直拍型

完整展示打开包装、商品全貌和细节，配合音乐、字幕或解说。适合包装有价值、礼盒、配件丰富、外观精致和新品；核心价值依赖使用效果时只能作为辅助内容。

### 买家卖家秀型

人物或身体部位作为主体，真人实测、上身、上手、使用或讲解。适合服饰、鞋、珠宝、美妆、发型工具以及需要真人效果或尺寸参考的商品。

### 测评对比型

出现至少 1 个同类产品，或同一商品的不同口味、款式、规格进行客观对比，不能明显偏袒。适合同质化、功能可测、参数可比和购买前会横向比较的商品。

## 画像修正

- `requires_model = true`：买家卖家秀 +1。
- `has_clear_function = true`：卖点讲解 +1。
- `requires_large_scene = true`：生活场景 +1。
- `suitable_for_comparison = true`：测评对比 +1。
- `style_sensitive = true`：买家卖家秀 +1、生活场景 +1。
- `product_has_packaging = true`：开箱 +1。
- `is_high_price = true`：测评对比 +1、卖点讲解 +1。
- `is_electronic = true`：卖点讲解 +1、测评对比 +1。

## 限制

- 开箱不能只因有包装自动成为最高优先级。
- 无真人素材时，买家卖家秀只能标记推荐但材料缺失，不能执行。
- 无竞品或对比对象时，测评对比只能标记推荐但材料缺失，不能执行。
- 无真实场景素材时，生活场景型可执行，但必须生成场景拍摄清单。
