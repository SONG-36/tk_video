# 型号级视频批次规划页

- 页面目标：为一个目标型号规划 3—5 条差异化测试视频及 A/B/C 生成等级。
- 输入字段：`batch_id`、批次目标、视频数量；每条视频的分型、风格、卖点、场景、痛点、等级、提示词开关、素材需求、镜头数、时长。
- 输出数据：带 `product_series_id + variant_id + batch_id` 的 `video_batch_plan`。
- 通过条件：ID、数量、每条必填项齐全；图片支撑等级；不满足条件者不能进入 A 级；用户确认。
- 状态依赖：创意确认后进入，通过后为 `video_batch_planned`。默认 `1A_1B_1C`，最多五条。
